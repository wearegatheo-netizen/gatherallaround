🎵 BGM 동기화 설정

Suno 플레이리스트의 곡들을 사이트 BGM에 자동 추가하는 북마클릿을 만듭니다.

1 GitHub Personal Access Token 입력

토큰은 브라우저에만 저장되며 서버로 전송되지 않습니다. 새 토큰 발급 →

2 플레이리스트 ID 입력

https://suno.com/playlist/여기-ID-부분

3 북마클릿을 북마크 바로 드래그

↓ 이 버튼을 북마크 바로 드래그 해주세요

📌 BGM 동기화

북마크 바가 안 보이면 ⌘ Shift B (Mac) / Ctrl Shift B (Win)

위 정보를 입력하면 북마클릿이 생성됩니다

4 사용 방법

  1. Suno 플레이리스트 페이지 (suno.com/playlist/...) 열기
  2. 플레이리스트가 화면에 모두 로드될 때까지 잠시 대기 (스크롤로 모든 곡 노출)
  3. 북마크 바의 📌 BGM 동기화 클릭
  4. 새 곡이 자동으로 music.json에 추가되고 사이트에 반영
⚠️ 이 북마클릿에는 GitHub 토큰이 포함되어 있습니다. 본인의 북마크 바에만 저장하고, 절대 공유하지 마세요.
